‚ö° Key Principle: Leadership is inferred only from event windows (e.g., 0.02% in 2s). Absolute price differences do not imply lag.

Core Data Contracts

Contract Definition
Session One Polymarket 15-minute market window: start_ms, time_left, winner, trades
Significant Move Event Clustered triggers across sources within MOVE_WINDOW_MS; includes leader and per-source lags
OB Slope Change in order-book imbalance over fixed window (60s). Sign=direction; magnitude=confidence
Dislocation Binance(mid) - Chainlink(price), $ and %. Used as context/filter, NOT standalone
Trading Logic

Decision Engine

Default SKIP. Trade only when ALL binary checks align.

‚òë
EDGE Checklist (All Must Pass)
  1. 1
    DATA HEALTH
    Chainlink + Polymarket fresh; 2+ exchanges fresh
  2. 2
    SIGNIFICANT MOVE EVENT
    Detected using canonical threshold/window (0.02% in 2s)
  3. 3
    LEADER IDENTIFIED
    >=2 sources triggered; leader != Chainlink
  4. 4
    POLY BEHIND GATE
    Dislocation >= $X OR slow reprice detected
  5. 5
    PRESSURE CONFIRMS
    |OB_slope| >= Y and direction matches leader
  6. 6
    ENTRY WINDOW VALID
    Fixed delay passed; spread/slippage caps pass
Trade Rule
IF all_six_gates == YES THEN TRADE
ELSE SKIP // with reason code
